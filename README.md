# ASPNET-dk25ttc1-tranhuuloc-WebBDS
This Website i create to manage and view residential properties around Ho Chi Minh City, VietNam

Features:
 1. User
    - Register/Login Account
    - Profile management
    - Change/Forgot Password
    - Bookmarks
 2. Property List
    - View products (properties)
 3. Search System
    - Prices
    - Area
    - Types
    - Locations
 4. Posts Management
    - For middlemans allow them to create/edit their posts
      
#Tech use:
 - Typescript + CSS + Javascript
Authencation:
 - JWT
 - Refresh Token
 - Bcrypt
Storage:
 - SQL + Cloudinary
